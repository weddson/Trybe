IP 
Internet Protocol

Um IP possui 4 campos de dígitos separados por pontos. São chamados de endereço lógico.

TCP
Transmission Control Protocol(Protocolo de Controle de Transimissão)

ISP
ISP ou Internet Service Provider é o provedor de internet.

DNS
DNS ou Domain Name System é um dispositivo com uma base de dados distribuída que gerencia os nomes de serviços, computadores ou qualquer dispositivo conectado à internet.

Port Number
O número de porta é um número de 16 bits utilizado para identificar uma porta específica no servidor, e está sempre associado ao endereço IP.

Host
O Host é qualquer dispositivo conectado à rede que pode transmitir ou receber pacotes IP.

HTTP
Hyper-text Transfer Protocol é o protocolo utilizado para a comunicação entre navegadores e servidores na internet.

URL
É a sigla para Uniform Resource Locators e identifica um recurso web específico.

HTML
Hyper text markup language

CSS
Cascading Style Sheets
